三、认识HTML

1、HTML是什么
(1)HTML 不是一种编程语言，而是一种标记语言，由一些标签组成
(2)HTML 使用标记标签来描述网页，一个Html文档就是一个网页
2、HTML标签
(1)HTML 标记标签通常被称为 HTML 标签 (HTML tag)
(2)HTML标签是由尖括号包围的关键词，比如<html>
(3)HTML标签通常是成对出现的，比如<body>和</body>标签对中的第一个标签是开始标签，第二个标签是结束标签；也有单标签，比如<br />、<img />
3、模板认识
(1)DOCTYPE声明
(2)html、head、body标签
(3)编码集、三要素 utf-8 === utf-8 gbk/gb2312 === 系统默认 或者 ANSI

四、基础标签的认识

<div></div>
Division 最常见标签，网页中用来做整体结构的一个长方形区块；
通过css来控制区域的宽度、高度、背景等等样式。
<p></p> <span></span>
p、span标签通常被用来放入文字；
p标签，可设置宽高的文字区块；
span标签，通常用来组合文字，来设置单独样式；
可以但不限于放入文字。
<br />
换行标签
<hr />
横线换行，水平分割线
<del></del>
文本删除线，不常用
<u></u>
文本下划线，不赞成使用
<strong></strong> <b></b>
放在其中的文字会被加粗；
strong标签的强调程度较强，用于SEO优化；
b标签是单纯的视觉加粗效果；
可以但不限于放入文字。
<em></em> <i></i>
放在其中的文字会变成斜体；
strong标签的强调程度较强，用于SEO优化；
<a></a>
超链接标签，用于给页面中的内容添加超链接；
<a href=" " target="">内容</a>； href属性规定点击后所跳转的页面；
target属性规定新页面打开方式
　　_self 当前窗口打开
　　_blank 新窗口打开
　　_new 同一新窗口
锚点标签，用于给同一页面内，内容间的跳转创建锚点链接；
<a href="#锚点名"></a> <a id="锚点名"></a>
特殊字符
空格: &nbsp; 大于号: &gt; 小于号: $lt; 引号: &quot; 版权号: &copy; ……

五、框架知识

<frameset></frameset>
用来定义一个框架集；
rows 定义行(取值 px / % / * )
cols 定义列(取值 px / % / * )
不能和 <body> 一起使用
<frame /> -- 表示框架中的某一个部分
src 显示的网页的路径
name 框架名
frameborder 边框线(取值 0 / 1)
scrolling 滚动条(取值 yes / no / auto)
<noframes></noframes>
不支持框架的浏览器
frmaeset的用法：
<frameset> <frame src="" /> <frame src="" /> <frame src="" /> <noframes> <body>内容</body> </noframes> </frameset>
<iframe>
css3标签,允许和 body 一起使用
width 宽（取值 px / %）
height 高（取值 px / %）
name 框架名
frameborder 边框线（取值 0 / 1）
scrolling 滚动条（取值 yes / no / auto）
src 显示的网页的路径
